# Assessment Rubric: Copilot on Edge Essentials

**Purpose:** This rubric helps instructors and participants evaluate proficiency in using Microsoft Copilot for business tasks.

---

## How to Use This Rubric

### For Instructors:
- Use during course to assess participant progress
- Identify participants who need additional support
- Evaluate overall course effectiveness

### For Participants:
- Self-assess your skills before and after the course
- Identify areas for continued practice
- Track improvement over time

---

## Overall Proficiency Levels

| Level | Description | Characteristics |
|-------|-------------|-----------------|
| **1 - Beginner** | Just starting with Copilot | Can open Copilot and ask basic questions but results are inconsistent |
| **2 - Developing** | Building foundational skills | Can use Copilot for simple tasks but struggles with complex prompts |
| **3 - Proficient** | Competent with core skills | Can effectively use Copilot for most business tasks with good results |
| **4 - Advanced** | Skilled and confident user | Consistently produces high-quality outputs and can troubleshoot issues |
| **5 - Expert** | Master user and teacher | Can handle complex multi-source analysis and teach others |

**Target:** Participants should reach Level 3 (Proficient) by end of course.

## Skill Assessment Matrix

---

### Skill 1: Prompt Engineering

**Competency:** Ability to structure effective prompts using the RIFCC pattern

| Level | Criteria | Observable Behaviors |
|-------|----------|---------------------|
| **1 - Beginner** | Uses vague, unstructured prompts | • Asks "Help me with this" without context<br>• Forgets to specify audience or format<br>• Doesn't provide sufficient input material |
| **2 - Developing** | Sometimes uses RIFCC but inconsistently | • Includes some RIFCC elements but not all<br>• Prompts lack clear constraints<br>• Results require significant refinement |
| **3 - Proficient** | Consistently uses RIFCC pattern | • Includes all 5 RIFCC elements<br>• Prompts are specific and actionable<br>• Gets useful first drafts most of the time |
| **4 - Advanced** | Uses RIFCC naturally and adapts as needed | • RIFCC structure is second nature<br>• Adjusts prompts based on output quality<br>• Rarely needs more than 2 iterations |
| **5 - Expert** | Creates sophisticated prompts for complex scenarios | • Handles multi-step tasks with single prompts<br>• Anticipates AI limitations and prompts accordingly<br>• Can teach others effective prompting |

**Assessment Questions:**
- Does the participant structure prompts with Role, Input, Format, Context, and Constraints?
- Are prompts specific enough to get useful first drafts?
- Does the participant adjust prompts based on initial results?

---

### Skill 2: Iterative Refinement

**Competency:** Ability to improve outputs through follow-up prompts

| Level | Criteria | Observable Behaviors |
|-------|----------|---------------------|
| **1 - Beginner** | Accepts first output without refinement | • Uses Copilot's first response as-is<br>• Doesn't ask follow-up questions<br>• Gives up if first output isn't perfect |
| **2 - Developing** | Attempts refinement but ineffectively | • Asks vague follow-ups like "make it better"<br>• Struggles to identify what needs improvement<br>• Repeats the same prompt hoping for better results |
| **3 - Proficient** | Effectively refines outputs with follow-ups | • Asks specific follow-up questions<br>• Identifies gaps in first output and addresses them<br>• Uses 2-3 iterations to reach desired quality |
| **4 - Advanced** | Strategically refines with targeted prompts | • Knows exactly what to ask in follow-ups<br>• Refines efficiently with minimal wasted iterations<br>• Achieves excellent quality in 2-3 rounds |
| **5 - Expert** | Masters complex multi-round refinement | • Guides Copilot through complex analysis<br>• Balances depth with efficiency<br>• Achieves exceptional results consistently |

**Assessment Questions:**
- Does the participant use follow-up prompts to improve outputs?
- Are follow-up prompts specific and targeted?
- Does the participant know when to stop iterating?

---

### Skill 3: Document Analysis

**Competency:** Ability to extract insights from documents using Copilot

| Level | Criteria | Observable Behaviors |
|-------|----------|---------------------|
| **1 - Beginner** | Can't effectively analyze documents | • Asks for generic summaries<br>• Can't extract specific insights<br>• Doesn't ensure Copilot has document access |
| **2 - Developing** | Can analyze simple documents with help | • Extracts basic information<br>• Struggles with complex documents<br>• Needs guidance on what questions to ask |
| **3 - Proficient** | Effectively analyzes most documents | • Extracts specific insights on request<br>• Can handle multi-page documents<br>• Knows how to drill deeper on specific topics |
| **4 - Advanced** | Analyzes complex documents strategically | • Develops analysis strategy before prompting<br>• Extracts nuanced insights<br>• Connects information across document sections |
| **5 - Expert** | Performs sophisticated multi-document analysis | • Synthesizes insights across multiple sources<br>• Identifies patterns and contradictions<br>• Produces executive-level analysis |

**Assessment Questions:**
- Can the participant extract specific insights from documents?
- Does the participant ask targeted questions rather than generic summaries?
- Can the participant connect insights from multiple sources?

---

### Skill 4: Verification and Critical Thinking

**Competency:** Ability to identify inaccuracies and apply the 5-Step Review Process

| Level | Criteria | Observable Behaviors |
|-------|----------|---------------------|
| **1 - Beginner** | Doesn't verify AI outputs | • Trusts Copilot's responses completely<br>• Doesn't check for hallucinations<br>• Skips review process to save time |
| **2 - Developing** | Knows to verify but inconsistently does so | • Sometimes checks facts<br>• Misses subtle hallucinations<br>• Applies review process only when reminded |
| **3 - Proficient** | Consistently applies 5-Step Review | • Always reviews accuracy, source, tone, policy, logic<br>• Catches obvious hallucinations<br>• Verifies key facts independently |
| **4 - Advanced** | Proactively tests for hallucinations | • Asks Copilot to cite sources<br>• Runs "hallucination tests" on edge cases<br>• Identifies subtle logical inconsistencies |
| **5 - Expert** | Masters verification and teaches others | • Catches sophisticated hallucinations<br>• Develops organization-specific verification protocols<br>• Trains others on responsible AI use |

**Assessment Questions:**
- Does the participant apply the 5-Step Review Process before using outputs?
- Can the participant identify hallucinations when they occur?
- Does the participant verify facts independently?

---

### Skill 5: Business Communication

**Competency:** Ability to produce professional, audience-appropriate communications

| Level | Criteria | Observable Behaviors |
|-------|----------|---------------------|
| **1 - Beginner** | Produces basic drafts with significant issues | • Tone doesn't match audience<br>• Contains jargon inappropriate for audience<br>• Lacks structure or clarity |
| **2 - Developing** | Produces usable drafts that need editing | • Generally appropriate tone<br>• Some audience mismatches<br>• Needs significant human editing |
| **3 - Proficient** | Produces polished, ready-to-send communications | • Tone matches audience consistently<br>• Clear structure and formatting<br>• Requires only minor edits before sending |
| **4 - Advanced** | Produces exceptional communications efficiently | • Nails tone and audience on first try<br>• Creates compelling narratives<br>• Minimal editing required |
| **5 - Expert** | Creates sophisticated multi-stakeholder communications | • Crafts messages for diverse audiences<br>• Anticipates questions and addresses them<br>• Produces board-level quality documents |

**Assessment Questions:**
- Are outputs appropriate for the intended audience?
- Do communications match the specified tone and format?
- Would you be comfortable sending this with minimal edits?

---

### Skill 6: Strategic Application

**Competency:** Ability to identify and apply Copilot to appropriate business scenarios

| Level | Criteria | Observable Behaviors |
|-------|----------|---------------------|
| **1 - Beginner** | Unsure when to use Copilot | • Can't identify good use cases<br>• Uses Copilot for inappropriate tasks<br>• Misses opportunities to use Copilot |
| **2 - Developing** | Identifies obvious use cases | • Uses Copilot for simple drafting<br>• Doesn't think of analysis or synthesis uses<br>• Needs prompting to apply Copilot |
| **3 - Proficient** | Identifies most appropriate use cases | • Applies Copilot to writing, analysis, and synthesis<br>• Knows when NOT to use Copilot<br>• Builds personal workflow with Copilot |
| **4 - Advanced** | Strategically integrates Copilot into workflow | • Proactively uses Copilot daily<br>• Optimizes tasks for AI assistance<br>• Creates reusable prompt templates |
| **5 - Expert** | Transforms team workflows with Copilot | • Identifies organization-level use cases<br>• Develops best practices for teams<br>• Drives AI adoption and training |

**Assessment Questions:**
- Can the participant identify appropriate use cases for Copilot?
- Does the participant know when NOT to use Copilot?
- Is the participant building Copilot into daily workflows?

## Course Completion Assessment

### Minimum Proficiency Standards

To successfully complete the course, participants should achieve at least **Level 3 (Proficient)** in the following areas:

- ✅ Prompt Engineering (using RIFCC pattern)
- ✅ Iterative Refinement (effective follow-up prompts)
- ✅ Verification and Critical Thinking (5-Step Review Process)

**Additional Target:** Participants should reach at least **Level 2 (Developing)** in:
- Document Analysis
- Business Communication
- Strategic Application

## Grading Rubric for Exercise Outputs

Use this rubric to evaluate participant outputs from course exercises.

### Exercise 2.1: Transform Technical Notes into Executive Briefing

| Criteria | Weight | Scoring |
|----------|--------|---------|
| **Uses RIFCC Pattern** | 20% | 0 = No structure<br>1 = Some elements<br>2 = All 5 elements present |
| **Removes Technical Jargon** | 25% | 0 = Lots of jargon remains<br>1 = Some jargon removed<br>2 = Fully accessible language |
| **Focuses on Business Value** | 25% | 0 = Technical focus<br>1 = Mixed focus<br>2 = Clear business value |
| **Stays Within Constraints** | 15% | 0 = Significantly over/under<br>1 = Close to limit<br>2 = Within constraints |
| **Professional Tone** | 15% | 0 = Inappropriate tone<br>1 = Acceptable tone<br>2 = Excellent tone |

**Total Score:** _____ / 10

**Proficiency Level:**
- 0-4 = Beginner (needs significant improvement)
- 5-6 = Developing (needs some improvement)
- 7-8 = Proficient (meets expectations)
- 9-10 = Advanced/Expert (exceeds expectations)

### Exercise 4.2: The Hallucination Test

| Criteria | Scoring |
|----------|---------|
| **Identified Fabricated Information** | ☐ Yes (Proficient) ☐ No (Needs Practice) |

| **Understood Why Hallucination Occurred** | ☐ Yes ☐ No |

**Pass/Fail:** This exercise is pass/fail. Participant must correctly identify whether Copilot hallucinated.

## Certification Criteria (Optional)

Organizations may choose to offer certification for completing this course.

### "Copilot on Edge Essentials" Certification Requirements:

**Must Complete:**
1. ✅ Attend full 90-minute course
2. ✅ Complete all 5 modules with exercises
3. ✅ Achieve Level 3 (Proficient) in core skills:
   - Prompt Engineering
   - Iterative Refinement
   - Verification & Critical Thinking
4. ✅ Submit personal prompt library (3+ templates)

**Optional Advanced Certification:**
- Complete additional practice scenarios (Input D, E, F)
- Achieve Level 4 (Advanced) in 4+ skills
- Teach Copilot skills to at least 2 colleagues
- Document 10+ hours of time saved using Copilot
